Serving Pi

The Raspberry Pi Zero W

Do you need a basic experimental linux-based web server to play around with at home? Well I have the perfect solution for you! The Raspberry Pi Zero W. As you will see from the specs below, its no power house. But it can get the job done if you're just needing a simple HTML server with FTP access that you are not putting out there on the internet for everyone to use (see Connecting Via FTP).

Low Power Consumption

It is a very compact little guy that runs off of a Wifi connection, and uses an insignificant amount of power at only about 1 watt when its active. Looking at the power consumption of a typical computer easily being close to 100 watts, the Raspberry Pi is much cheaper solution for an experimental server, a huge savings if left on all the time.

It's Small!

This little guy, in the case, only takes up 0.5 X 1.5 X 3.2 inches. That's a wonderfully insignificant amount of space in your already cramped office! Once you have it all setup, and can SSH into it from any computer on the network (see Connect Via SSH), all you need is a power cable running to it. That's less space than your phone plugged into the charger.

It's Portable

Since it only uses about one watt of power when it is active, you could run it off of a battery pack quite easily. If you would like to take it to work or school with you. The only other consideration would be that you have a wifi router with you to access it from your laptop. And you would probably want to configure it at home for any other wifi passwords that it may need (see Configuring The Wi-fi), unless you want to drag the other equipment with you to be able to connect to it directly.

Its Cheap

The other nice thing about it is it is really cheap. Micro Center has a kit for $12.48 currently that includes the Pi Zero W, a 16 GB microSD card, and a case to protect it. You will still need a power supply, and a few things during setup only. A mini-HDMI to HDMI cable and a HDMI monitor, a mini-USB hub or adapter to connect your keyboard, and the keyboard of course (no mouse is required as we are only installing a command line operating system to keep it more efficient). Current pricing for it all is less than $50 if you already have a keyboard and monitor you can use during setup.
